Seção de Imagens Adaptável

Demonstração da nova funcionalidade onde a moldura se ajusta automaticamente ao tamanho da imagem

Como Funciona

A nova seção de imagens adaptável permite inserir imagens de qualquer tamanho e a moldura se ajusta automaticamente, mantendo o estilo visual consistente com o tema do site.

Principais Benefícios:
  • Moldura se ajusta automaticamente ao tamanho da imagem
  • Inserção simples via HTML
  • Responsivo para todos os dispositivos
  • Mantém a identidade visual do site
  • Efeitos hover suaves e elegantes

Uso Básico

Para usar a seção de imagens adaptável, basta usar a seguinte estrutura HTML:

<div class="adaptive-image-container"> <div class="adaptive-image-frame"> <img src="caminho-da-imagem.jpg" alt="Descrição da imagem"> <div class="adaptive-image-caption">Legenda opcional</div> </div> </div>

Variações de Estilo

A seção oferece três variações de estilo para diferentes necessidades:

Estilo Padrão

Moldura com borda verde e fundo gradiente sutil.

<div class="adaptive-image-frame"> <img src="imagem.jpg" alt="Descrição"> </div>

Estilo Sutil

Moldura com borda mais discreta e fundo transparente.

<div class="adaptive-image-frame subtle"> <img src="imagem.jpg" alt="Descrição"> </div>

Estilo Elevado

Moldura sem borda com sombra mais pronunciada.

<div class="adaptive-image-frame elevated"> <img src="imagem.jpg" alt="Descrição"> </div>

Responsividade

A seção de imagens é totalmente responsiva e se adapta a diferentes tamanhos de tela:

Características Responsivas:
  • Desktop: Exibição completa com todos os efeitos
  • Tablet: Molduras se ajustam ao espaço disponível
  • Mobile: Imagens empilhadas verticalmente para melhor visualização
  • Imagens múltiplas: Automaticamente reorganizadas em coluna única no mobile

Pronto para Usar!

A seção de imagens adaptável está pronta para ser integrada ao seu site. Basta incluir o CSS e usar a estrutura HTML demonstrada acima.
