Seção de Imagens com Lightbox

Demonstração da nova funcionalidade onde a moldura se ajusta automaticamente e as imagens podem ser visualizadas em tela cheia

Funcionalidades

Moldura Adaptável

Se ajusta automaticamente ao tamanho da imagem

Lightbox Integrado

Clique na imagem para visualizar em tela cheia

Responsivo

Funciona perfeitamente em todos os dispositivos

Estilo Consistente

Mantém a identidade visual do site

Como usar o Lightbox:
  • Clique na imagem para abrir em tela cheia
  • Clique novamente na imagem ou no fundo escuro para fechar
  • Use o botão X no canto superior direito para fechar
  • Pressione ESC no teclado para fechar

Uso Básico

Para usar a seção de imagens com lightbox, basta usar a seguinte estrutura HTML:

<div class="adaptive-image-container"> <div class="adaptive-image-frame"> <img src="caminho-da-imagem.jpg" alt="Descrição da imagem"> <div class="adaptive-image-caption">Legenda opcional</div> </div> </div>
